Challenge 5: Accessibility Concerns

Some challenges for the visually impaired can include the visual chessboard and having it be audibly described. A form might also be hard to describe if there are label tags not on every field. Another challenge can be spacing between text, and can be audibly read incorrect. Having div tags and packages can aid in spacing between text and lines can also help with visually impaired userability.
